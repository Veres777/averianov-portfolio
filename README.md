# Averianov Portfolio

Jednoduchý webový projekt vytvořený ve Flasku jako osobní portfolio.

## Funkce
- Zobrazení informací o autorovi
- Seznam projektů
- Dovednosti
- Kontaktní formulář s uložením do databáze
- Admin sekce s přihlášením a zobrazením zpráv

## Technologie
- Python 3
- Flask
- Flask-WTF
- Flask-Login
- SQLAlchemy
- Heroku (nasazení)
